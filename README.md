- 파일 예시: `/src/ans/kitae0522/index.ts`, `/src/ans/kitae0522/problem.ts`
- PR Title 예시: `[송기태] - 교육생 테스트 1차 제출` or `[송기태] - 교육생 테스트 최종 제출`
